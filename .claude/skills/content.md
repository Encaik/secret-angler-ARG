# content skill — 撰写游戏叙事

撰写符合游戏世界观和角色设定的网站内容。

## 输入参数
- **内容类型**：community_post / user_update / article / gang_record / member_message / ending_text
- **所属阶段**：1-5
- **视角角色**：linyu（林屿）/ shenci（沈辞）/ gang（团伙）/ member（会员）/ neutral（中性）
- **内容长度**：short（1-3句）/ medium（1-3段） / long（完整文章）

## 输出格式
输出符合人设语气的中文文本内容。

## 角色人设速查

### 林屿（玩家视角）
- **身份**：大二学生，天文与未知现象深度爱好者
- **性格**：谨慎细腻、观察力极强、习惯记录各类异常线索
- **语言风格**：理性克制，喜欢用数据和坐标说话，偶尔流露对宇宙的敬畏
- **口头禅/常用词**："记录一下"、"值得注意"、"坐标显示"、"信号异常"
- **探秘偏好**：外星目击、太空异常信号、NASA数据、射电望远镜

### 沈辞（失踪舍友）
- **身份**：大二学生，古代遗迹、小众古文明爱好者
- **性格**：大胆执着、观察力敏锐、擅长发现场景细节破绽
- **语言风格**：热情洋溢但不失严谨，实地考察式叙述，爱用手绘图
- **口头禅/常用词**："实地探访"、"现场痕迹"、"遗迹结构"、"手绘记录"
- **探秘偏好**：冷门古迹、古文明遗址、摩崖石刻、墓葬结构

### 团伙成员（内部视角）
- **语言风格**：冷漠、功利、术语化（"猎物"、"布景"、"场次"、"清理"）
- **禁忌**：不直接使用"绑架"、"囚禁"等字眼，用暗语替代

## 规范约束

- 所有"超自然现象"必须在后续揭示为人工骗局
- 线索文本暗示而不明示
- 匹配小众爱好者社区的语言风格（不要太像商业文章）
- 碎片化：每段文本只提供部分信息，驱动玩家继续探索
