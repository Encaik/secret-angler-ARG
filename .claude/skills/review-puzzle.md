# review-puzzle skill — 谜题审查

审查谜题的可解性和难度合理性。

## 输入参数
- **谜题 ID**：要审查的谜题标识符

## 审查维度

### 1. 可解性（是否能在不借助外部提示的情况下解出？）
- [ ] 线索是否足够？玩家需要的最小线索数是否满足？
- [ ] 线索是否在正常的探索路径上？还是藏得太深？
- [ ] 解密工具是否普通玩家可以获取？（浏览器自带 / 搜索引擎可找到）

### 2. 公平性（解法是否唯一且合理？）
- [ ] 答案是否唯一？（不能有多个合理答案）
- [ ] 解码路径是否清晰？（不能莫名其妙跳到正确答案）
- [ ] 是否依赖过于冷门的知识？

### 3. 难度评级
- **初级**：10分钟以内可解，仅需观察力+常识
- **中级**：10-30分钟，需要了解基础解密（Base64/凯撒）
- **高级**：30分钟以上，需要多步推理+工具使用

### 4. 叙事融入度
- [ ] 谜题是否与剧情有关联？
- [ ] 解密后获得的信息是否有叙事价值？
- [ ] 是否只是"为解密而解密"的机械操作？

## 输出格式

```markdown
## 谜题审查报告：[谜题ID]

**可解性**：✅ 可解 / ⚠️ 有风险 / ❌ 不可解
**公平性**：✅ 公平 / ⚠️ 可能多解 / ❌ 存在歧义
**难度评级**：初级 / 中级 / 高级
**叙事融入度**：✅ 紧密 / ⚠️ 一般 / ❌ 脱节

**具体意见**：
- ...
**改进建议**：
- ...
```
