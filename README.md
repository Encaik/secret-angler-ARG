# 🎣 秘境垂钓者 (Secret Angler)

> 一款以网站为载体的浏览器端 ARG 解谜游戏。

## 关于游戏

《秘境垂钓者》是一款虚实结合的 ARG（Alternate Reality Game）解谜游戏。玩家如同普通访客般浏览一个看似真实的"探秘者联盟导航站"，通过搜索、查看源码、解密隐藏内容、追踪线索，逐步揭开网站背后地下灰产团伙的真相。

### 核心体验

- **网站即游戏**：不需要下载任何客户端，游戏就是一个真实的网站
- **自主搜索解密**：无直白提示，线索藏在页面源码、图片隐写、控制台消息中
- **分层隐藏**：公开页面 → 源码隐写 → 关键词触发 → 隐藏后台，难度递进
- **多结局**：基于你的探索深度和关键选择，触发四种不同结局

### 四种结局

| 结局 | 触发条件 | 结果 |
|------|---------|------|
| 🌑 沉沦囚笼 | 探索不足，独自前往 | 接替舍友成为新的直播素材 |
| 🌓 孤影救赎 | 救出舍友，未收集完整证据 | 二人逃离，黑产继续运作 |
| 🌖 微光有限 | 救出舍友 + 部分证据 + 报警 | 捣毁当前基地，核心团伙潜逃 |
| 🌕 秘境破晓 | 100% 探索 + 完整证据链 | 彻底覆灭灰产链条，全员获救 |

## 技术栈

- **框架**：[Astro](https://astro.build/) 静态站点生成器
- **语言**：TypeScript
- **样式**：纯 CSS（复古小众社区风格）
- **部署**：GitHub Pages（`docs/` 目录）

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
# → http://localhost:4321

# 构建到 docs/（GitHub Pages）
npm run build

# 预览构建产物
npm run preview
```

## 项目结构

```
secret-angler-ARG/
├── src/
│   ├── pages/          # Astro 页面（每个 .astro 对应一个路由）
│   ├── components/     # 可复用组件
│   ├── scripts/        # 客户端 TypeScript（谜题、进度、结局）
│   ├── data/           # 游戏内容数据
│   ├── styles/         # CSS 样式
│   └── assets/         # 静态资源
├── game-design/        # 游戏设计文档
├── docs/               # 构建产物（GitHub Pages 部署目录）
└── public/             # 直接映射到根路径的文件
```
