📡 关于探秘者联盟导航站

探秘者联盟导航站成立于2021年，是一个由探秘爱好者自发维护的非营利性社区站点。

我们的使命

为探秘爱好者提供一个纯净的信息交流平台。没有商业广告，没有营销号洗稿，没有算法推荐。只有最原始的探秘热情和最真实的经验分享。

🙋 加入我们

本站由一群业余探秘爱好者共同维护。我们没有固定的办公室，有人用周末维护服务器，有人用探秘途中的空闲时间审稿。如果你也热爱探秘，欢迎加入我们的团队。

📋 当前招募岗位

🗺️ 探秘线路投稿人

如果你在探秘过程中发现了值得分享的点位——无论是未记录的岩画、信号异常区、还是冷门古迹——欢迎投稿。优先收录偏远、人迹罕至的秘境，让更多同好知道这些隐秘的角落。

要求：至少完成过 3 次独立探秘行程
📝 内容审核员（深度探秘方向）

负责审核探秘投稿的真实性与专业度。需要对探秘领域有较深理解，能辨别投稿中的不实信息或抄袭内容。深度探秘级别的审核员可访问站点历史档案库和专属资料线路，用于交叉验证投稿内容。

要求：注册满一年，发布过至少 5 篇原创探秘内容
🔧 服务器运维志愿者

协助维护站点服务器和数据库。需要熟悉加密通讯协议，因为部分内部管理功能通过安全通道访问。有暗网/Tor 网络经验者优先——仅为技术需求，非用于非法用途。

要求：2年以上Linux运维经验，了解 .onion 站点部署
🌐 社区联络员

负责与其他探秘社区保持联系，寻找"有潜力的新成员"加入我们的圈子。如果你在其他平台（户外论坛、B站、知乎等）上发现过认真做探秘内容的人，欢迎引荐——我们会发送定向邀请。

要求：活跃于多个探秘/户外圈子
