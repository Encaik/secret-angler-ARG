明天去贺兰山，简单记录一下准备情况

作者：地下三尺 · 发布于 2026-05-25 02:13 · 阅读 478 · 评论 12

这学期课少，总算能多跑几个点了。明天出发去贺兰山，简单记录一下。

目标是之前有人在论坛上提过的一批岩画，不在公开文物图录里。查了坐标，在银川西侧，离市区大概三个小时车程，最后一段路没公路，得靠腿。看卫星图的话，最近的水源地差不多一公里外，扎营得提前把水备足。

装备

  • 35L包 + 防雨罩（这个季节山里说不定有阵雨）
  • 三季帐 + 5°C睡袋
  • 老GPS手持机，用顺手了也不想换别的
  • 头灯 + 备用电池两组
  • 急救包 + 蛇药（宁夏蝮蛇多，这个不能省）
  • 三天干粮 + 净水片
  • 速写本 + 炭笔，岩画记录用
  • 充电宝两块（山里大概率没信号，但相机和GPS得续着）

路线

银川出发，西夏区方向进山，沿干沟徒步大概六公里，翻一个海拔1800左右的垭口，到岩画所在的那面崖壁。看别人的攻略说沿途有明显的牧道，应该不会太难走。计划三天两夜，把主崖面和旁边两条侧沟都仔细走一遍。

昨晚查了几家天气预报，接下来一周应该都是晴。不过山里的事情说不准，还是把硬壳塞包里了。

顺路看看

对了，导航站最近推了好几个贺兰山那边的点位。有几个描述写得还挺详细的——岩画风格、GPS坐标、甚至手绘示意图都有——正好这次去的区域跟其中两个重合，打算顺路验证一下。如果靠谱的话，以后可以多参考站里的推荐。

之前也去过几个他们推的点，怎么说呢，描述有对有错吧。有些确实可以，有些写得有点夸张。不过业余爱好者维护的站，也不能要求太严。总体来说当个参考还行。

对了，站里最近更新了公告。看内容就是普通的季度维护通知——会员升级、帖文归档、搜索优化什么的。不过措辞挺有意思的，跟一般探秘社区的公告风格不太一样。有空的话可以去读一读。

回来之后我会把这次的手绘图和照片整理出来。贺兰山的岩画在全国范围内都算特别的——主要是凿刻技法，跟南方的彩绘不一样。希望能有一些新发现。

另外，@signal_屿——之前聊的事我整理了一些东西，去看私信。
