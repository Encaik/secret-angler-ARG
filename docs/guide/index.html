🧭 探秘者指南

必备装备清单

  • GPS定位设备 + 纸质地图（山区信号不稳定）
  • 头灯 + 备用电池（夜间探秘必备）
  • 急救包 + 常用药品
  • 充电宝（至少20000mAh）
  • 多功能刀具
  • 防水冲锋衣 + 登山鞋
  • 便携频谱分析仪（信号探测专用）
  • 相机/手机 + 三脚架（记录用）

安全须知

  • 行前报备：告知至少一位朋友你的目的地和预计返回时间
  • 结伴出行：不建议独行探秘，尤其偏远区域
  • 天气检查：出发前确认目的地未来48小时天气
  • 尊重当地：进入村庄、遗址前与当地人沟通，遵守当地规则
  • 不留痕迹：带走所有垃圾，不破坏任何遗迹
  • 如果不对劲，立刻回头：好奇心是探秘的动力，但安全永远是第一位

推荐工具

  • 奥维互动地图 — 离线卫星图
  • Star Walk 2 — 天文观测辅助
  • RTL-SDR — 便携信号扫描
  • 两步路 — 户外轨迹记录
