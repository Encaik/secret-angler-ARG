归源宗 · 寺规（节选）

一、单次法会后冷却90日，同一站点不得连续猎取。违者逐出僧团。

二、不问来处，不究归处。执行层只负责接引与护送，不问供体去向。中层以上可查阅完整档案，但不得向执行层透露。

三、叛者必诛。凡泄露僧团信息或企图脱离者，由维那执行处置。家属不牵连——除非家属构成威胁。

四、禁私吞功德金。所有供养收入入供养库（老田），任何成员不得私下收取买家款项。

五、禁伤及无辜。外勤行动只针对目标本人。随行人员、路人、当地居民——不得碰。违者等同于叛。

六、遇险自决。若外勤行动暴露或被执法部门拦截——各自脱身，不得牵连寺院。被抓者自行承担，寺院概不承认关系。家属可由香灯酌情抚恤。

※ 以上寺规由宗主（老周）于2021年制定，首座（老韩）修订。违者由维那（铁头）执行处置。
