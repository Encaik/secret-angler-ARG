叛变过程

以下为事后复盘。此人转变并非一朝觉悟——乃是逐层崩溃。

第一层：自我合理化（2024.11 — 2025.06）
此人加入组织之初，将自身角色框定为"技术外包"。即便察觉论坛站点的推荐内容存在系统性造假（虚假古迹描述、伪造信号报告），也自圆为"营销手段"——"哪个网站不编内容？反正也没人真会去。"他刻意不去想那些去了的人的下场。每当疑虑浮现，他便用"我还有房贷、有孩子要养"来压下去。

第二层：被迫目击（2025年夏·第一次外勤）
组织以"现场设备调试"为由将其派往外勤。他被告知"只是去架信号设备和通讯"。在伏击点目睹了行动的全程——目标被从背后按住、麻醉、拖上车。事后领队告诉他："你的手也在上面了。这些设备是你架的。通讯是你调的。你觉得自己能撇清吗？"他没有回答。回程的车上他一直看着窗外。

此后他开始有意识地回避关于"目标去向"的信息。他负责维护服务器、清理日志、管理域名——但他刻意不去看"在押人员区"和"商品列表"的数据表。这是一种精心维护的自我欺骗："我不看，就不知道。不知道，就不是共犯。"

第三层：镜像崩溃（2026年1月·第二次外勤）
第二次行动。目标是一名年轻女性。在按住她的过程中，她的手机从口袋滑出，屏幕朝上落在泥土里。屏保照片是——她和一位老妇人的合影，两人都笑得很灿烂。

那老妇人的长相——瘦削的脸、花白的短发、笑起来眼角向下弯——和他两年前死于癌症的母亲一模一样。

他说他当时手开始抖。他说那个女孩被拖上车的时候，他听到她在叫"奶奶"。

那晚回到住处后，他第一次翻看了那些他一直逃避的数据表。他看到了"库存"栏、"品级"栏、"估价"栏。他看到了自己第一次参与接引的两个人的最终去向——一个"已化"，一个"已渡"。他盯着屏幕上的字看了很久，然后吐了。

他终于让自己理解了：之前那两个人不是被"转移"了。是被拆开卖了。

第四层：暗中反击（2026年2月）
他没有选择报警。他说他查过了——他自己也是共犯，刑期不会短。而且组织知道他妻子和孩子的地址。

他选择了一条更慢、更稳妥的路：利用自己的技术权限，在两个月内秘密收集了所有能接触到的证据数据——服务器日志、基地坐标、暗网数据截图、直播频道地址、加密货币流水——然后写了那个自动举报脚本。30天不登录系统，自动发送。多收件人部署，无法全部拦截。

在收集证据期间，他也翻到了宗主的背景资料。宗主（老猫/老周）的妻子于2021年因肝癌去世——这正是归源宗开始运作的同一年。郑义在他的笔记里写了一段话："老周说他做这一切是为了他女儿小雨。他说如果没有钱，你最爱的人就会死在医院走廊上，而你只能看着。我说那你有没有想过，你现在赚的每一分钱——都有另一个人的家人死在走廊上。他没回答。我也不知道我希望他怎么回答。"

他在他的日记里写了一段话（后被组织从云端删除，但部分缓存恢复了）：

"我知道我做了什么事。我也知道即使把这些东西发出去，那些人也活不过来。我做这些不是因为我觉得自己可以被原谅——我没什么可被原谅的。

我只是不想在死的时候，脑子里还是那个老妇人的脸。

那不是我妈。但可以是任何人的妈。"

附件D：组织成员侧写（郑义个人记录）

以下为郑义在收集证据期间记录的组织核心成员背景与个人动机。这些记录散见在他的私人笔记中，部分已被组织删除，此版本为缓存恢复。

宗主（老周 / 论坛ID：老猫）
退休地理教师。妻子2021年肝癌去世——没钱移植，死在医院走廊里。这是他做这一行的起点。不是图钱——虽然现在确实有钱了。他的驱动力是恐惧：害怕哪天自己倒了，女儿落到跟他妻子一样的下场。女儿叫小雨，是他密码的来源，也是他唯一还像个人的部分。有时候开会开到一半他会接女儿的电话，声音一下子就变了——从一个犯罪组织头目变回了一个普通父亲。那一刻你能看到他有多分裂。

首座（老韩）
前军队后勤，少校衔。因为一次训练事故——不是他的责任但背了处分——被劝退。有个先天残疾的儿子，需要昂贵的终身治疗。老韩是组织里最让我困惑的人。他不残忍，甚至算得上公正。定规矩，守规矩。他打仗的时候应该是个好军官——但现在他的"战场"是绑架和贩卖人口。他对自己说的话是："上面定目标，我负责执行。跟我在部队里做的没有区别。" 区别是——部队的目标是军事设施，这里的目标是人。

监院（阿坤）
比我小三岁，和我一样做技术。大专没读完就出来了——不是笨，是没钱。有个妹妹在读大学，学费和生活费都是他在供。他妹不知道他做什么——以为他在科技公司上班。每次视频前他都会换一件有公司logo的卫衣，那是他以前从一家真公司顺回来的。他设计的监控系统可以在受害者完全不知情的情况下上传GPS、步数、心率——技术上非常漂亮。他跟我说过："如果我用这些技能去干合法的，可能已经在大厂了。" 我说你现在也可以去。他没回答。

铁头（行者领队）
坐过五年牢。故意伤害。出来之后找不到任何正经工作——有案底的人，连快递站都不要。他家里有个老娘，住在乡下，以为他在城里工地干活。他每个月寄钱回去，数额不小，他妈应该也猜到这钱不干净——但没问。铁头是这个组织里唯一让我感到害怕的人。不是因为他暴力——他当然暴力——而是因为他已经不再为自己的暴力感到不安了。对他来说这就是一份工作，跟搬砖没区别。

小周（行者副手）
二十出头，刚入伙三个月。是铁头带上来的——据说是一个"朋友的朋友"介绍的。这个"朋友"多半也是圈内人。小周到现在都不知道自己到底在干什么——他只被安排开车、望风、搬东西。但贺兰山那一单他看见了。回来之后他哭了。铁头跟老韩说这个人扛不住。老韩说再观察。我猜小周的结局不会好——这种组织里，扛不住的人通常只有两条路：要么被培养成扛得住的人，要么被处理掉。中间状态太危险了，因为他们会犹豫，而犹豫会害死所有人。

马姐（典座 / 运输）
前货运公司老板。破产之后带着三辆没被银行收走的货车入了伙。跑遍三省所有县道，知道每一个查报站、每一个不用过磅的小路。她丈夫跟人跑了，自己带着两个儿子。大儿子在南京上大学，小儿子还在读高中——两个孩子的学费都从这里出。马姐是这个组织里最"正常"的人——她从不碰现场，从不看货，只管方向盘和油表。但她不是不知道自己在运什么。有一次她跟老田吵架，老田说"物流只是成本，不是业务"——她的回答是："没有我的车轮子，你们的业务连省都出不了。"

赵医生（外护 / 医疗对接）
前急诊科医生。因为一次医疗事故被吊销执照——病人家属闹得很大，医院让他背了锅。他现在对接流动手术团队——就是那些从外省飞过来做器官摘取的"飞刀"医生。他把摘取器官叫"手术"，把受害者叫"供体"，把杀人叫"终止生命维持"。有一次他跟我说："我做了十五年急诊，见过太多人在走廊上等死等不到器官。现在这些器官能救活别人——你告诉我，如果没人做这件事，这些器官会去哪？埋进土里？" 我没回答。因为我当时在想：他说的是对的，但他对吗？

老田（香灯 / 资金）
前银行职员。因为小额金融诈骗被开除，从此没办法在金融业立足。现在管组织的钱——加密货币、现金流转、买家付款、工资发放。他快六十了，无儿无女，这是他攒的养老钱。老田从不碰现场，从不看货。他说他的工作只是"管理数字"。有一次我问他——你看着那些数字的时候，会想到它们对应的是什么吗？他想了一下说："数字不流血。这是我选择看数字的原因。" 这是他对自己说过的最诚实的话。

若你得此信——说明我已经不在了。这个脚本按我设定的方式触发了。挺好。至少有一件事按计划走了。

附件中是我在过去三个月里收集的所有我能拿到的东西。服务器日志、坐标数据、交易记录。够立案了。也许够定罪了。我不懂法律，但我知道这些是真的。

如果你需要核实——找"深空古迹探秘导航站"这个论坛。搜索：深海裂隙、星门坐标、布景基地。你会找到他们没来得及清理干净的东西。

受害者被关在鄂西一个废弃萤石矿里。坐标在附件中——31.12, 111.45。还有人活着。至少发信这一刻还有人活着。

另外注意——这个组织有一个伪装成爱好者论坛的网站叫"深空古迹探秘导航站"。看起来是个寻常的探秘爱好者社区。实际上那是他们筛选目标的陷阱。如果你在上面看到什么"秘境推荐"——不要去。那不是旅行指南。那是伏击点。

我知道你收到这封信之后可能会想查我是谁。我叫郑义。今年三十八岁。以前是做IT的。失业之后被人拉进了这个组织。

我不是好人。我帮他们抓过人。两个。

第一次的时候他们跟我说——"只是去架设备"。我去了。我架了。人被抓的时候我站在旁边看。后来领队跟我说——"你的手也在上面了。"他说的是实话。我的确在。

第二次的时候我已经知道了会发生什么。我还是去了。因为我怕如果我说不去——他们会对我家人做什么。也因为我一直在骗自己：只要我不去看那些人的下场，我就不算真正的共犯。这是谎话。我知道。

让我停下来的是第二次接引的那个女孩。她手机掉出来的时候——屏幕亮着。屏保是她和她奶奶的照片。那位老太太长得很像我妈。

我妈是两年前癌症走的。走的那天我在医院陪她。她最后跟我说的是——"别太累。早点睡。"

那个女孩被拖上车的时候，我听到她在叫"奶奶"。

那晚我翻了那些我一直没敢看的文件。我看到了前两个被抓的人后来怎么样了。我对着屏幕吐了好久。然后我就知道——我骗不下去了。

所以这些证据不是我"良心发现"送出来的。是我的呕吐物。是我终于没办法再咽下去的那些东西。

我不求你原谅我。我不需要原谅。我做这些也不是为了赎罪——赎不了。

我只是想在我死的时候，脑子里不要再是那个老妇人的脸。

那不是我妈。但可以是任何人的妈。


—— 郑义（代号：深观），2026年2月28日绝笔
