标准操作流程（内部文件）

最后修订：2026-01-15 · 密级：内部 · 仅限核心人员

⚠ 本文件为组织核心操作流程。任何未经许可的外传将导致严厉处置。前成员郑义（深观）曾试图泄露本文件——已于2026年3月被处置。以此为戒。

一、目标筛选（2-12周）

目标：在论坛中识别潜在目标，建立信任。

方法：

  • 以普通爱好者身份（如"外联人员"账号）参与论坛讨论
  • 发布真实、有用的帖文建立信誉（路线分享、装备评测等）
  • 观察活跃用户的：发帖频率、出行偏好、是否独行、社会关系广度
  • 为每位潜在目标填写目标评估量表（见附录A）

筛选优先级：

  • ✓ 优先：独行者、偏好偏远地点者、社会关系少者、非本地人、无军警/政府背景者
  • ✗ 排除：多人同行者、社会关系广泛者、军人/警察/政府人员

二、目标评估

目标：从候选目标中选定1-2人，确定为下一轮行动对象。

评估维度：

  • 独行倾向（1-10分）：是否惯于独自出行？
  • 偏远偏好（1-10分）：是否偏好偏远、信号差的探秘地点？
  • 社会关联（1-10分）：失踪后有多少人会持续追问？
  • 专业知识威胁（-10到0分）：是否有能识破我们伪造内容的专业能力？
    ⚠ 注：沈辞的专业背景（古迹/历史）评分威胁-6——他有能力识别我们编造内容中的漏洞。但他的偏远偏好和独行倾向极高。综合评估后仍列为目标。这是一次高风险决策。

三、兴趣引导

目标：根据目标的兴趣偏好，在论坛推送匹配的探秘地点信息。

引导类型：

  • 古迹类 → 偏远古迹/未记录遗址（需配合我们编造的内容）
  • 天象类 → 信号热点/异常现象频发区（需配合信号灯阵列）
  • 自然类 → 未开发秘境/隐秘景点（无需额外布置，纯粹靠地点偏远）

核心原则：推荐的地点必须是真实存在的偏远地点。不需要伪造地点——伪造成本高且容易穿帮。我们只需要目标到达该地点。剩下的由行动组处理。

⚠ 组织软肋：我们在古迹/历史/天文等专业领域的编造内容经常有漏洞。已被多位有专业背景的用户识破（包括沈辞）。新修订流程要求：遇到专业知识过高的目标，优先排除而非冒险行动。沈辞的行动是一次教训——他的历史学知识使他提前识破了我们的引导内容，差点暴露整个组织。

四、诱导出行

目标：让目标"自己发现"地点并产生前往的意愿。

  • 通过论坛推送、评论区暗示、私信"热心推荐"等方式引导
  • ⚠ 关键：让目标觉得是自己找到的——而非被引导
  • 如果目标犹豫，可由外联人员私信提供"独家消息"或"结伴邀请"来推动
  • 目标决定出行后，外联人员可提供看似有益的出行建议（路线、装备）来确认其具体行程

五、现场行动

目标：在偏远地点安全、隐蔽地完成行动。

  • 至少提前 48小时 到达地点附近埋伏
  • 行动组：2-3人（1人望风、1-2人执行）
  • 装备：麻醉枪、通讯设备、伪装车辆（科考/施工外观）
  • 目标到达后确认身份 → 确认周边无目击者 → 迅速行动
  • 行动完成后立即清理现场：撤除埋伏痕迹、回收信号灯（如有）
  • ⚠ 野外的行动痕迹最难清理——篝火、食物残渣、烟蒂、装备碎片。需两人交叉检查。

六、善后处理

目标：最小化事后暴露风险。

  • 行动完成后12小时内删除论坛上与该地点相关的所有引导帖文
  • 目标最后的帖文保留不删（删除反而可疑）
  • 外联人员继续在论坛正常活动，观察舆论
  • 如有质疑帖文，由外联人员以"普通用户"身份进行舆论引导
  • 进入至少三个月冷却期——期间禁止新行动

站点废弃准则：

  • 行动 2-5 人后废弃当前论坛站点
  • 废弃方式：停止更新、停止回复，让站点自然"沉寂"
  • ⚠ 绝不突然关闭——会引起受害者家属和执法部门注意
  • 新站点使用全新域名、全新数据库、全新"站长"身份

案例：沈辞（代号：三尺）
独行倾向：9 · 偏远偏好：9 · 社会关联：7 · 非本地性：8 · 专业知识威胁：-6 · 网络声量：2

最终决定：列入目标。结果：行动成功，但他在出行前留了加密证据给林屿——这就是低估他专业能力的后果。
