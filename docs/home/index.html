各位探秘者：

站点计划于本周六（6月7日）凌晨进行服务器迁移，预计维护时间2:00-6:00。维护期间站点可能间歇性无法访问。不便之处敬请谅解。

另外几件事：

一、近期发现部分探秘点位的投稿描述与实际地貌存在出入。如果你实地探访后觉得某条内容存在问题，请在原帖下留言说明，帮助后来的探秘者获得更准确的参考信息。

三、最近两个月注册的新用户比较多，欢迎各位。如果是通过邀请码进来的，记得去新人报到区发个帖介绍一下自己——大家都在等你的故事。

四、版主招募持续进行中。如果你注册满一年、发过至少5篇原创探秘内容，且对某个探秘领域有深入理解，欢迎去"加入我们"页面提交申请。
