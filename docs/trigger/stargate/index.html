🔧 信号伪造技术说明（内部文档片段） 缓存于 2026-05-28

设备选型：

  • 主选：HackRF One + 功放模块（成本低、频段宽、可编程）
  • 备选：LimeSDR（用于1420MHz氢线频段的精细调制）
  • 供电：太阳能板 + 12V铅酸电池组（无人值守可持续运行2-4周）

信号设计原则：

  • 信号特征：窄带载波 + 不规则的脉冲调制（听起来"像是有信息"但无法解码——因为没有信息）
  • 频段选择：优先使用SETI关注的频段（1420 MHz氢线、1662 MHz羟基线），增加被业余射电天文学家发现的概率
  • ⚠ 局限：非天文学专业人士用频谱仪一拉就能看出调制模式不像自然天体。此方案对有信号分析经验的人无效——但目标群体中这类人极少。

部署位置选择：

  • 与"古迹秘境"推荐点位的地理坐标高度重合——因为两套方案共用同一批伏击地点
  • 距离最近公路至少 5 公里，确保目标需徒步进入
  • 信号覆盖半径约 1-3 公里（受地形和功放功率限制）
