2026-03-12

有人问川西那条线——甘孜到色达的县道去年塌方过，今年还没完全修好。要走的话建议绕道炉霍，多花半天但安全得多。

2026-01-08

这个站现在人少了很多。前年还挺热闹的。不过我这种老头子也不在乎热闹，清静也好。反正也不是为了热闹才建这个站的。

2025-12-20
