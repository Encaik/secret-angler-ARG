# 主线流程

> 最后更新：2026-06-04｜综合优化版·七阶段渐进式解谜
> 用途：跟踪主线是否可走通，每次游戏内容修改后必须同步更新本文档
> 关联文档：`剧情推进逻辑.md`（叙事自洽）、游戏代码/页面（实际内容）——三方必须同步
> 设计原则：密室逃脱式解谜体验，观察力>知识储备，每谜题≥3条独立提示路径

## 核心机制

### 游戏循环

```
探索页面 → 发现异常 → 形成假设 → 搜索验证 → 获得线索 → 解锁新区域 → 探索页面...
    ↑                                                                      ↓
    └──────────────────── 理解升级，重新审视旧信息 ←────────────────────────┘
```

### 关键设计规则

- **页面 = 近期内容**（近3-6个月帖文/动态），直接浏览可见
- **搜索 = 历史存档**（2025年及更早），线索分散在可检索到的老帖中
- **页面内闭环**：所有谜题线索在正常浏览范围内可发现，不依赖F12/源码/外部工具
- **搜索是核心串联机制**：关键词可触发隐藏页面跳转，也可检索历史存档拼接线索
- **线索碎片化**：单个线索不构成完整答案，需要多源交叉验证
- **删帖痕迹 = 隐藏线索**：得手后团伙删除点位推荐和引导评论，被删内容的"空缺"是线索
- **未得手者的吐槽 = 独立情报源**：去过点位但未被伏击的人发的"假点位"帖
- **钓鱼站与内部站分离**：公开钓鱼站与团伙内部运营中枢是两套独立系统；进入内部站时应有"换站感"
- **宗教化伪装层**：团伙内部系统（第三层 `/hidden/`）在视觉上伪装为虚构佛教宗派"归源宗"的寺庙法务管理系统——暖色调界面、佛教称谓与术语。这是犯罪团伙的"内部黑话"层，所有犯罪活动均被宗教语言包裹。玩家需看穿宗教外衣，将其还原为犯罪实质。核心术语对应：金刚舍利=骨骼交易、甘露=器官交易、金身渡化=整人贩卖、法身示现=活体展示、接引仪轨=作案流程手册、因缘海=暗网交易市场、善缘榜=市场挂牌区、因果文牒/因果卷=证据。**所有谜题密码、搜索关键词、URL 路由均不受此伪装层影响——伪装仅限于页面视觉与文本措辞。**
- **核心罪行 = 器官贩卖+人口贩卖**：暗网直播是"商品展示窗"，真正主营业务是器官和人口交易

### 7.2 四大结局

#### 结局1：孤身沦陷（坏结局）

**触发**：证据收集不足（0-6分），或触发"闯入贺兰山腹地"

- 屏幕渐黑。"游戏结束？不——这只是开始。"

#### 结局2：孤军救人（中立结局）

**触发**：证据7-11分，选择了报警但证据不够完整

**叙事**：
- 林屿集齐了基地薄弱点位线索

- "你做了一切能做的——但这不够。有些人的失踪，连证据都没留下。"

#### 结局3：局部破获（良结局）

**触发**：证据12-17分

**叙事**：

#### 结局4：全链覆灭（完美结局）

- 全部受害者（含此前站点）获救——"旅人"在跨境转运前被拦截
- 团伙全员落网——包括境外头目（国际刑警红色通缉令）

- 沈辞和林屿在警局做笔录——沈辞："我就知道你会查到的。"
