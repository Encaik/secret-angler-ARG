# 主线流程

> 最后更新：2026-06-03
> 用途：跟踪主线是否可走通，每次游戏内容修改后必须同步更新本文档

## 核心机制

- **页面 = 近期内容**（近3-6个月帖文/动态），直接浏览可见
- **搜索 = 历史存档**（2025年及更早），线索分散在可通过搜索检索到的老帖中
- **页面内闭环**：所有谜题线索必须在正常浏览范围内可发现，不依赖F12/源码/外部工具
- **搜索是串联游戏的核心**：关键词可触发隐藏页面跳转，也可检索历史存档拼接线索

## 第三阶段：沈辞的加密信息

### 3.1 沈辞私信箱 `/inbox/`
- 登录 shenci 后访问私信箱，看到对话的另一半——来自林屿（signal_屿）的消息

- **引导方向：** 去沈辞主页看私密草稿
- **需验证：** shenci 登录后私信箱内容切换为林屿的来信

### 3.2 沈辞主页私密草稿 `/user/shenci/`
- 登录 shenci 后，页面底部出现🔒私密草稿区域（仅账号本人可见）
- 草稿内容：
  - **绿色代码块**：`dmtsaXcgeWRveGggbHYgd2t1aGggbmhienJ1ZyBsdiBnaGhzIHVsaXc=`
  - **解码提示1**："外层是一层很常见的编码方式——计算机之间传数据经常用的那种"（暗示Base64）
  - **解码提示2**："字母全都往后偏移了固定位数——我习惯用那个数，林屿老说我'什么事都绕不过三'"（暗示Caesar ROT3）
  - **指向**："把两层都解开之后，会得到一个中文词，用站里的搜索框搜一下就能打开一扇门"
- **需验证：** 未登录shenci时草稿不可见；登录后草稿正确显示；Base64字符串完整可复制

### 3.3 解码
- **第一层 Base64：** 解码绿色字符串 → `vkliw ydoxh lv wkuhh nhbzrug lv ghhs uliw`
- **第二层 Caesar ROT3：** 每个字母向前移3位 → `shift value is three keyword is deep rift`
- 得出关键词 → **深海裂隙**
- **辅助线索：** 搜索"加密"可在历史存档中找到林屿关于Base64和凯撒密码的讨论帖
- **需验证：** Base64解码结果正确；ROT3解密结果正确

### 3.4 搜索触发 `/trigger/rift`
- 在搜索框输入"深海裂隙" → 触发跳转
- **需验证：** 搜索"深海裂隙"是否跳转到 `/trigger/rift`
